# Minimal Image Lightbox

A vanilla JavaScript and CSS image lightbox with smooth zoom transitions and backdrop.

## Features

-   **Zero Dependencies**: No jQuery or external libs.
-   **Smooth Transition**: Fade and scale animations.
-   **Keyboard Support**: Close with `ESC` key.

## Installation

1.  **Clone the repository** OR copy the files.
2.  **HTML Usage**:
    ```html
    <link rel="stylesheet" href="css/lightbox.css">
    <script src="js/lightbox.js" defer></script>
    ```

## Usage

Add `class="lightbox-trigger"` to any image you want to open in the lightbox.

```html
<img src="image.jpg" class="lightbox-trigger" alt="My Image">
```
